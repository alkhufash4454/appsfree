# تطبيق سوداني (Sudani App)

تطبيق أندرويد احترافي مبني بلغة **Kotlin** و **Jetpack Compose** للتعامل مع خدمات سوداني.

## المميزات:
- تسجيل دخول عبر **OTP**.
- عرض الرصيد والنقاط.
- عرض الخدمات النشطة بتصميم دائري.
- تفعيل عروض (300 ميقا، 1 جيجا).
- تجميع النقاط تلقائياً.
- تصميم ليلي (Dark Mode) احترافي.
- متصل بـ API سوداني مباشرة.

## كيفية البناء:
1. افتح المشروع في **Android Studio**.
2. انتظر تحميل التبعيات (Gradle Sync).
3. اضغط على **Run** لتشغيل التطبيق على المحاكي أو جهاز حقيقي.

## البناء التلقائي:
يحتوي المشروع على ملف **GitHub Actions** في المسار `.github/workflows/android.yml` للبناء التلقائي عند الرفع على GitHub.
